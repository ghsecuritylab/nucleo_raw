Webserver на базе lwIP TCP/IP стека
Текущая страница    Передача данных    Прием данных

     Этот веб-сервер предназначен для демонстрации примера обмена данными платы STM32F4-DISCOVERY с компьютером на базе стека lwIP TCP/IP. В качестве контроллера физического уровня сети использована плата DP83848 (PHY).
     Популярность интерфейса Ethernet связана, прежде всего, с возможностью передавать данные на большие расстояния при высокой помехоустойчивости, что подтолкнуло разработчиков оборудования к производству большого количества различных технических и программных средств для расширения возможностей Ethernet.
